settings.json

disable_ai: true + hide tab bar, toolbar, minimap, all side panels. Project panel 200px with git dots, gitignore hidden. Verified keys, ready to paste.

tasks.json — the agent launcher

One task per agent: tmux new -A -s agent-indx claude, with cwd per repo/worktree, spawned into the center grid. ⌘1–9 bound to your portfolio. Adding an agent = 5 lines.

keymap.json

Terminal-context bindings so TUIs keep their keys, split/focus-move on ⌘D/⌘⌥-arrows, plus Claude's /terminal-setup once for shift+enter.

Gotcha 1 — Zed kills terminals on quit

Layout + cwd restore; processes don't. That's why every agent runs inside a named tmux session — relaunch Zed, panes reattach to still-running agents.

Gotcha 2 — git panel is single-repo

Multi-root works, but Zed's git panel shows one repo at a time. Mitigation: lazygit pane (⌘4) — better than Zed's panel anyway.

What v0 ships

The agent cli repo = these 3 config files + an install script + README. Done = you open one Zed window and run 3 agents across 2 repos, and they survive a Zed restart.
